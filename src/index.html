Projects

The Odin Book - in development

This project is a full-stack web application that utilizes JavaScript and its various libraries and frameworks to build a social media platform. The back end of the application is built using Node.js and Express, with Passport.js utilized for user authentication. The RESTful API endpoints are created with Express to facilitate communication between the front end and the back end.


On the front end, the application is built using React and styled components, which provides a dynamic and visually appealing user interface. The front end interacts with the back end through the RESTful API endpoints, and the data is stored and managed in a MySQL database.

Weather App 2.1

The creation of this project required proficiency in Vanilla JS and Axios for hitting multiple API endpoints, including OpenWeather for current weather forecasts and 5-day/3-hour forecasts, as well as TheZipCodes for latitude/longitude data based on zip codes.


The project also employed ES6 await syntax for asynchronous functions and promise interactions, utilized a functional programming approach to limit application side effects and improve testability, and implemented TDD principles for non-dom based logic, including API calls. The project was further validated through the use of Github actions to create consistent testing and vetting conditions for all commits and pull requests.

Pokemon Memory Game

The Memory Game is a simple browser-based matching game built using React, HTML, CSS, and JavaScript. The game presents a grid of cards and with a unique image, and the player must chose a unique card on each click to raise their score. If they click on a card that they have chosen previosly then their score is reset and the must start again.


React is used to build the user interface and handle the game logic, such as tracking the state of the cards and checking for matches. The components in the game are built using React, providing a highly interactive and dynamic user experience. HTML and CSS are used to provide an intuitive and visually appealing interface for the player.

Battle Ship

This repository is a JavaScript (ES6+) battleship game featuring OOP using factory functions and test-driven development with Jest. Players can attack the enemy gameboard and the game ends when one player's ships are all sunk. The UI allows for rendering gameboards and taking user input for attacking.

Express Blog Api

This project is a backend RESTful API created to provide backend functionality to a client-side blog application.


The API is built using javascript, express, and mongoDB and provides API endpoints for GET, POST, PUT, and DELETE requests, each of which interacts with a database to persist user blog data.

Library App 2.0

The TOP Library 2.0 is a web-based library application built using JavaScript, Node.js, Express, and MongoDB. The application allows users to search, view, and manage a library of books, with features such as adding new books, editing book details, and checking out books.


The application utilizes Node.js and Express for the back-end, providing a RESTful API for managing the library data stored in a MongoDB database. The front-end is built using JavaScript, providing a dynamic and interactive user interface for the library.


Overall, the TOP Library 2.0 is a comprehensive and user-friendly library management system, showcasing the power of JavaScript and its related technologies in building web-based applications.
