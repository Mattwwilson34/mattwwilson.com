FEATURED PROJECTS

BLOG API

An API only back end built with node, express, mongoDB, and mongoose paired with a react front end. All data is stored in mongoDB and passed to the front-end via fetch calls to the custom backend. User auth was implemented using json web-tokens as well as react state and browser local storage to persist user login.

GAME AND SEEK

A react front end coupled with a firebase backend that allow users to play a photo tagging game based on the nintendo gamecube. Users are timed as they search a large image of many characters. After all characters are found the timer stops, the firebase stores is updated with the players time, and the player is redirected to the leaderboard.

TEXTILES-R-US

A react front end that allows users to search a catalog of various textiles. Users are able to add various textiles to their shopping cart and when done shopping can see a checkout page with a list of all of the items that were added to their cart with the option to update quantaties or delete items from the cart entirely.

BATTLESHIP 2021

A remake of the 1931 board game that was build with test driven development. The game allows users to place their battle ships on their board. Once all ships are placed a computer board will appear allowing the user to begin attacking. User and computer take turns until all of one players ships are sunk.
